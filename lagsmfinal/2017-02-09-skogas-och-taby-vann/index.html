Skogås och Täby vann

Skogås skrällde borta mot Fyrisfjädern i Uppsala. 3-4 blev resultatet efter Golden set. Matchen Halmstad - Täby slutade 2-4.

Skogås vann alla tre matcher med damer inblandade. Och när det var dags för Golden set var Carl Harrbacka/Irina Andersen starkare än Nico Ruponen/Tilda Sjöö. Skogåsparet vann med 11-7 och överraskningen var ett faktum.

I Halmstad saknades Felix Burestedt, medan Täby hade tvingats lämna Clara Nistad hemma. Resultatet blev 2-4 efter det att Täby vunnit samtliga herrmatcher.

Svenska Badmintonligan består denna säsong av sju deltagande lag, Umeå, Täby, Spårvägen, Kista, Skogås, Fyrisfjädern, Halmstad.

Antal delmatcher har gått från sju till sex delmatcher där herrdubbel 2 tas bort. Vid tre lika i matcher avgör Golden Set. Golden set innebär att vid tre lika i matcher spelas ett avgörande set till 11 i en match som båda lagen genom bortröstning av olika kategorier kommer fram till. Det kan innebära att en spelare som redan spelat två delmatcher går in och spelar Golden Set.
