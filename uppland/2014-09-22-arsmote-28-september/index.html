Årsmöte 28 september

Härmed inbjuder Styrelsen för Badminton Sweden Uppland samtliga sina medlemsföreningar till Årsmöte 2014

Söndagen den 28 september på Upplands BoService Arena (Fyrisfjädern) i Uppsala. Med start klockan 16.00.

 

BSU bjuder alla som kommer på mat i enklare form, samt erbjuder milersättning för resan till mötet.

 

Samtliga handlingar kommer att finnas på årsmötet.
