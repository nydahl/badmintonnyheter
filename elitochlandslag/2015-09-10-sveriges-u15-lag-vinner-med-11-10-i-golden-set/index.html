Sveriges U15-lag vinner med 11-10 i golden set!

Joel Hansson räddar två matchbollar

Idag startade Skånska Ungdomsspelen i Malmö och först ut var U15 för landskamp mot Finland. Det blev en jämn match från start mot det finska laget och efter att sista ordinarie match var klar stod det klart att vi spelat 3-3. Vilket innebar att det skulle spelas ett golden set i någon kategori. Efter diskussioner med laget och med Finland så blev HS den match som skulle spelas. Ett set till 11 utan förlängning. Joel Hansson tog på sig uppgiften och lyckades vända matchen till vinst efter att ha haft två matchbollar mot sig. Vinst till slut med 11-10 och därmed också vinst till Sverige med 4-3. 

 

Övriga resultat från landskamperna:

U15 Finland-Danmark 0-6

U15 Sverige-Danmark 0-6

U13 Sverige-Danmark 1-8
