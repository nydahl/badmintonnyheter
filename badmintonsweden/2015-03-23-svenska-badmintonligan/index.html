Semifinaler avgörs i två lagmatcher

- Om ett lag vinner båda matcherna, så är de vinnare.

- Vid lika antal vunna lagmatcher så räknas antal vunna delmatcher.

- Vid lika antal vunna delmatcher räknas antal vunna set.

- Vid lika antal vunna set räknas antal vunna poäng. 

- Om det fortfarande inte går att utse en vinnare sker lottning för att få fram en vinnare
