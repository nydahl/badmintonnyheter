Testperiod: Nytt räknesätt

Badminton World Federation (BWF) har under veckan skickat ut information om en ny testperiod för räknesätt. Det är nu tredje gången på några år som det testas. Den nya perioden kommer börja med start i september 2016 till siste december 2017.

Testen kommer att ske på vissa internationella tävlingar med status/nivå Grand Prix, challenge och serie, samt internationella juniortävlingar.

Det nya systemet innebär fortfarande 5 set till 11 med olika slut/förlängningar på seten.

Test variant 1: Vid 10-10, vinst vid två poängs försprång eller till max 15.

Test variant 2: Vid 10-10, blir det spel om först vunna 3 poäng. Det kan då bli 3-0, 3-1 eller 3-2 till någon spelare.

 

BWF har producerat en lista med arrangörer till tävlingar som kan ansöka om att testa det nya räknesättet.

BWF meddelade redan i Maj 2015 att ingen ändring i räknesättet skulle göras, men nya tester skulle genomföras. Efter kvalperioden till OS som slutar i början på Maj, så finns det naturligtvis möjligheter att agera igen utan att skapa alltför stora problem.

Vad är en fördelen med ett nytt räknesätt?

Många anser följande argument för en ändring.

-Kortare set ger tidigare spänning i seten, viktiga poäng kommer redan vid  ställningen 6-7-8.

-Kortare set kan skapa mer möjligheter till överraskningar/setvinster för den svagare spelaren.

-Matcherna idag är för långa. Under de senaste två åren har många av matcherna i de internationella tävlingarna blivit längre och längre, många upp till 70-90 min. Det nya räknesättet skulle ta ner tiden om man ser till antal spelade poäng.
