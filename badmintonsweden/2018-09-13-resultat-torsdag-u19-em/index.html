Resultat torsdag U19-EM

Sju matcher med svenskt intresse spelades under torsdagens U19-EM i Tallinn. Det blev vinst i fyra matcher.

Bild: Tim Foo ger instruktioner till Melker Z-Bexell/Tilda Sjöö (Foto: Badminton Europe).

Melker Z-Bexell/Tilda Sjöö spelade en bra mixed med bra samarbete, positionspel och rörlighet. Efter lite problem i första set blev det mer enkelt i andra set mot det tyska paret Aaron Sonnenschein/Leona Michalski och slutligen vinst med 26-24, 21-12.

Johan Azelius visade upp ett bra singel spel och var även fysiskt stark i tredje set i sin vinstmatch mot fjärdeseedade Christoffer Grimley från Skottland. Siffrorna blev 21-18, 12-21, 21-18.

Ashwathi Pillai/Tilda Sjöö visade upp en bra damdubbel och vann i två raka set mot det duktiga tyska paret Emma Moszczynski/Jule Petrikowski med 21-18, 21-16.

Joel Hansson/Melker Z-Bexell hade inga problem att vinna sin match mot det sjätteseedade slovenska paret Klemen Lesnicar/Domen Lonzaric med 21-11, 21-14.

Colin Hammarberg hade en sämre dag och förlorade mot Nicolas A. Mueller från Schweiz med 14-21, 16-21.

Även Malena Norrman förlorade sin singel, men var nära att ta andra set och då hade matchen kunna sluta annorlunda mot 6:e-seedade Maria Delcheva från Bulgarien. Det blev dock förlust i två set med 16-21, 23-25.

Ashwathi Pillai förlorade mot Amy Tan i två set med 10-21, 14-21.

Vinnarna spelar nu under fredagen åttondelsfinaler, och om de vinner dessa, även kvartfinaler under fredag eftermiddag och kväll.
