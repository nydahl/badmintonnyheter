Dålig dag för de svenska damerna

Sveriges damer förlorade med 1:4 mot Island.

Det blev som väntat en svår match mot Island…

 

DS1: Sophia Hansson höll jämna steg med Ragna Ingolfsdottir i första set och tjejerna kämpade länge om vem som skulle ta kommandot i matchen. Isländskan vann setet med knappa 22-20 och tog detta med sig till andra set, där hon ledde i stort sett hela vägen och till slut vann med 21-17 (0:1).

 

DS2: Jessica Carlsson var relativt undertippad mot Tinna Helgasdottir, men det var ändå Jessica som inledde bäst och hon lyckades ta en 17-13 ledning innan isländskan kom igång och vända underläget till setvinst med 21-18. Jessica kom starkt igen i andra set som hon vann med 21-11 genom fram för allt bra fight. Tredje set var jämnt fram till 12-13, men Jessica tappade överblicken i de avgörande bollarna varför setet till slut förlorades med 17-21 och därmed matchen (0:2).

 

DS3: Elinor Widh kämpade sig till seger i två raka set med 21-18 21-15. Växlade väl genomförde dueller och fina vinnarslag med lite för många enkla missar. Kändes ändå som att hon hade grepp om matchen från början till slut (1:2).

 

DD1: Emelie Lennartsson och Emma Wengberg kom dåligt in i matchen och tappade första set klart med 14-21. Spelet var aningen stressigt och även om tjejerna lyckades vinna andra set med 21-16 fick de aldrig in den rätta rytmen i spelet. Isländskorna spelade väldigt säkert och våra tjejer forcerade spelet för mycket och gjorde därmed lite för många misstag. Till slut förlust med 17-21 (1:3).

 

DD2: Amanda Högström och Cecilia Bjuner nådde inte heller upp på samma nivå som i gårdagens match och även här blev det förlust i tre set med 22-20 18-21 16-21 (1:4).

 

Tjejerna var självklart mycket besvikna efter matchen och framför allt för att vi förlorade utan att spela upp till vårt bästa.

 

Imorgon väntar Tyskland i den sista gruppspelsmatchen.
