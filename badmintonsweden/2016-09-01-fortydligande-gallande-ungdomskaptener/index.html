Lagledare/ Team manager     
Uppdrag vid speciella tävlingar enligt överenskommelse med TMU och FKU.
rollen innebär att hjälpa coacher och spelare med saker runt tävlingen så att coacher och spelare kan koncentrera sig på tävling. Tex vattenflaskor, mat, transporter, allmänt ”ta-om-hand”.

Rollen som FKU är vakant i U17 och U19.
Rollen som FTU och lagledare behövs alltid fler, och engagemanget sker i överenskommelse med TMU.

Dessa uppdrag föregås av utbildning/informationsträff med TMU för att vi skall hålla en gemensam linje, röd tråd, genom hela verksamheten.

För att inte missa någon som är intresserad, tar jag tacksamt emot intresseanmälan på dessa roller via mail.
