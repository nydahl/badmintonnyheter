Inbjudan till Smålandsligan 2019-2020

Inbjudan till Smålandsligan. Fyll i enkäten senast 18 aug.

Nu börjar det bli dags att planera upp Smålandsligan för säsongen 2019-2020. Vår ambition är att få till 5 omgångar på söndagar samma seriehelg som för div-1-4. Det hänger lite på att ni anmäler önskemål om att få arrangera hemmamatcher. Utan er hjälp kan vi inte genomföra alla omgångar.
